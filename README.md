# samara-web-course
Репозиторий для проектов студентов курсов Верстки в Самаре

Свой проект кладём в отдельнную папку в корне и именуем по фамилии, например
`/sobolev`

## Основные команды для работы с GIT
```
git clone
git pull
git commit
git push
```
